# Movie Ticket Booking App

A simple Python program that calculates the final price of movie tickets based on:

- User's age (eligibility for ticket and evening shows)
- Membership status (applies discounts)
- Extra charges for weekends or evening shows
- Service charges depending on seat type (Premium, Gold, Standard)

## Features

- Checks if the user is eligible for booking
- Calculates discounts for members
- Adds extra charges for evening or weekend shows
- Calculates service charges based on seat type
- Computes the final ticket price

## How to Run

1. Make sure Python 3 is installed on your computer.
2. Download or clone this repository.
3. Open the terminal/command prompt and navigate to the project folder.
4. Run the app with:
   ```bash
   python movie_ticket.py
